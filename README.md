# Gyn & Obs

Справочник по гинекологии и акушерству на React + Vite. В приложении есть поиск по нозологиям, карточки заболеваний, избранное, история просмотров, опросники и фармакологический модуль.

## Скрипты

- `npm run dev` запускает локальный Vite-сервер на `http://localhost:3000`
- `npm run build` собирает production-версию в `dist`
- `npm run preview` поднимает локальный просмотр production-сборки
- `npm test -- --watchAll=false` запускает тесты без watch-режима
- `npm run typecheck` проверяет TypeScript-типизацию без сборки

## Текущий стек

- React 19
- Vite 6
- TypeScript
- Framer Motion
- Bootstrap 5
- Testing Library + Jest через `react-scripts`

## Структура

- `src/App.tsx` содержит главный экран и фильтрацию данных
- `src/components` хранит модалки, навигацию, поиск и карточки
- `src/data` содержит медицинские справочники и фармакологические данные
- `src/hooks` хранит пользовательские хуки для темы, избранного, истории и debounce
- `src/types` содержит базовые типы приложения
